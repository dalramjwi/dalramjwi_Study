자바스크립트란?

프로그래밍 언어 중 하나인 자바 스크립트.

스크립트 언어의 표준인 ECMA 스크립트를 표준화한 스크립트 언어다.

  • 자바스크립트의 특징
    1. 객체기반의 언어이다.
    2. 함수형 프로그래밍 언어이다.
    3. 이벤트 주도형 프로그래밍 언어이다.
    4. 동적인 프로그래밍 언어이다.
  • 자바스크립트의 기초 문법
    1. 유니코드(UTF-8) 사용
    2. 대소문자 구분
    3. 단어(공백 포함) 구분
  • 자바스크립트의 키워드
    1. ECMA script 6 예약어
    2. 식별자 명명 규칙 - 알파벳, 숫자, 밑줄(_), 달러기호($)
  • 자바스크립트의 스크립트 변수
  • 자바스크립트의 스크립트 타입(자료형,데이터 타입)
      • 원시타입
    1. 숫자
    2. 문자열 (string)
    3. 논리형 (boolean)
    4. undefined
    5. null
    6. symbol
      참조타입
    1. 객체
    2. 함수
